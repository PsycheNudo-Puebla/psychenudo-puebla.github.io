# psychenudo-puebla.github.io
Nodo de difusión de psicología clínica, tecnología educativa y ética profesional en Puebla.

## Cómo contribuir

### Subir artículos
1. Crea un archivo Markdown en la carpeta `_articulos/` con el front matter requerido (ver ejemplo en `_articulos/tecnologia-clinica.md`).
2. Haz commit y push. GitHub Pages con Jekyll generará automáticamente la página.

### Foro de discusión
Usamos GitHub Discussions para dialogar con otros profesionales. Ve a [Discussions](https://github.com/psychenudo-puebla/psychenudo-puebla.github.io/discussions) para participar.

### Comentarios en artículos
Los comentarios se manejan vía Giscus, integrado con GitHub Discussions, respetando la privacidad.
